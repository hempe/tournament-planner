# Examples

Example implementations and integration demos.

## Files

### iframe-demo.html

Demonstration of embedding the Tournament Planner in an iframe within another website (Golf El Faro main site).

**Features demonstrated:**
- Full Golf El Faro page layout
- Tournament planner embedded via iframe
- Responsive iframe sizing
- Theme and navigation integration

**Usage:**
1. Start the tournament planner server:
   ```bash
   php -S localhost:5000
   ```

2. Open `http://localhost:5000/examples/iframe-demo.html` in a browser

3. The demo shows how the planner integrates with the main golf club website

**See also:** [Iframe Mode Documentation](../docs/IFRAME_MODE.md) for complete iframe integration details.
